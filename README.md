# CarTaxi 车辆租赁管理系统

## 项目简介
CarTaxi 是一个基于前后端分离架构的车辆租赁管理系统。当前阶段为第一阶段初始化版本，目标是完成项目骨架、文档、数据库初稿、后端基础框架与前端基础页面，为后续业务迭代打底。

核心需求来源于功能结构图，系统分为两大端：

- 用户端：个人中心、车辆投放地点、订单管理、归还管理
- 管理端：个人中心、用户管理、区域管理、车辆信息管理、车辆投放管理、车辆回收管理、订单中心、归还管理、管理员管理

## 技术栈
- 后端：Spring Boot 3、Maven、MyBatis-Plus、MySQL 8、JWT
- 前端：Vue 3、Vite、Element Plus、Pinia、Vue Router、Axios
- 数据库：MySQL 8

## 启动说明

### 1. 数据库初始化
1. 创建数据库：`cartaxi`
2. 执行脚本：`sql/cartaxi.sql`

### 2. 启动后端
在 `backend` 目录执行：

```bash
mvn spring-boot:run
```

默认端口：`8080`

### 3. 启动前端
在 `frontend` 目录执行：

```bash
npm install
npm run dev
```

默认端口：`5173`
