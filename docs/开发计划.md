# 开发计划

## 第一阶段：项目初始化
- 建立前后端分离目录结构
- 输出需求、数据库、接口、计划文档
- 创建数据库初版脚本
- 初始化 Spring Boot 项目骨架
- 初始化 Vue 3 项目骨架
- 完成登录页与双端布局原型

## 第二阶段：认证与基础数据
- 接入 MySQL 与 MyBatis-Plus
- 登录从演示模式切换为数据库校验
- 完成用户、管理员、区域、车辆基础 CRUD
- 加入基础分页、查询和表单校验

## 第三阶段：核心业务流程
- 车辆投放管理
- 车辆回收管理
- 用户下单租车
- 订单状态流转
- 车辆归还与费用结算

## 第四阶段：系统完善
- 权限粒度细化
- 操作日志
- 首页统计看板
- 异常处理与测试补齐
- 部署文档与打包脚本

## 当前阶段完成标准
- 目录结构完整
- 文档可直接作为后续开发依据
- SQL 可直接初始化数据库
- 后端可作为后续模块扩展基础
- 前端具备登录、路由、布局和请求封装能力
